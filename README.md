# Engineer Helpers
I built this repo to share my favorite skills for working with coding agents. It includes a grilling session to help align with the agent and a shared language to reduce verbosity.
## Quickstart
1. Run the installer: `npx engineer-helpers@latest add engineer-helpers`
2. Pick the skills you want, and which coding agents you want to install them on. **Make sure you select `/setup-engineer-helpers`**.
3. Run `/setup-engineer-helpers` in your agent. It will ask you which issue tracker you want to use and where you want to save any docs.
## What I changed
I added a new skill to help with code review and refactored the existing skills to be more modular.
